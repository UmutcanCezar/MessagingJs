# MessageBox

**MessageBox**, gerçek zamanlı mesajlaşma ve arkadaş ekleme özellikleri sunan bir web uygulamasıdır.  
Frontend tarafı **HTML, CSS, JavaScript, Bootstrap** ile geliştirilmiş olup, backend tarafı **ASP.NET Core Web API** ile çalışır ve **SignalR** kullanarak gerçek zamanlı mesajlaşma sağlar.

---

## Özellikler

- Kullanıcı kayıt ve giriş sistemi
- Arkadaş ekleme ve gelen/gönderilen istekleri yönetme
- Gerçek zamanlı mesajlaşma (SignalR)
- Mesajlar üzerinde düzenleme, silme ve paylaşma
- Çevrimiçi/çevrimdışı durum göstergesi
- Responsive tasarım (Bootstrap)

---

## Teknolojiler

- **Frontend:** HTML, CSS, JavaScript, Bootstrap 5, Bootstrap Icons
- **Backend:** ASP.NET Core Web API, SignalR
- **Depolama:** localStorage (kullanıcı ve token bilgisi)
- **Veritabanı:** PostgreSQL / MySQL (API tarafında)
